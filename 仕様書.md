## 友だち追加時の自動メッセージ

- Harness シナリオ「LINE Harness ウェルカム」（`friend_add`・即時配信）を使用
- 有効化: `pnpm tacteq:activate-welcome`
- 文面変更: 管理画面 → シナリオ → ステップ編集、または `WELCOME_MESSAGE` 環境変数で再実行

## データ移行オペレーション

### 既存友だちの同期

```bash
pnpm tacteq:sync-followers          # D1 の line_accounts からトークン取得
pnpm tacteq:sync-followers --dry-run
```

LINE API `followers/ids` でフォロー中ユーザーを `friends` テーブルに upsert。

### チャット履歴インポート（Chat Pro CSV）

1. LINE 管理画面でチャット履歴 CSV をエクスポート（ZIP 展開）
2. 友だち同期を先に実行
3. インポート:

```bash
pnpm tacteq:import-chat --dir /path/to/csv-folder
pnpm tacteq:import-chat --file sample.csv --inspect   # 列確認
pnpm tacteq:import-chat --dir ... --dry-run
```
